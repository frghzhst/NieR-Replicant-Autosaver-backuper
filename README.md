<h1 align="center">NieR Replicant(steam ver) saver</h1>
<div>
A tool for NieR Replicant(steam version) to save the save files and other game data incase of corruption.  
This tool creates a copy of the GAMEDATA file with a date.  
To restore a save, rename the save you want to restore from GAMEDATA_yyyy-MM-dd_hh-mm-ss to GAMEDATA and hit replace.  

-nd :  no program detection, won't automatically create a save of the save files and game data when you launch and close the game.  
-d : script will run continuously to watch for the game NieR Replicant to run, and automatically creates a save of the save files and game data when you launch and close the game.  

</div>
